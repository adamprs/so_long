# so_long

## Description

so_long is a small 2D game built using the MiniLibX graphical library.

The objective is to create a simple game where the player collects all collectibles on a map and then reaches the exit using the shortest path possible. This project focuses on window management, event handling, textures, and basic game logic.

The game uses a map file to define walls, collectibles, player position, and exit, and ensures that the map is valid and solvable.
